Help

Guidance

+ : Start Game

↻ : Reset Game

¡ : Information

× : Exit Game

✓ : Submit Answer

← : Delete Answer

How To Play

You need to press + to start the game each time. After pressing + game will show you some characters in random order and you must remember the exact order, and then use visual keyboard to enter them in previous order. Finally you must press ✓ to submit your answer. Be careful that you may only have 3 mistakes.

راهنما

+ : شروع بازی

↻ : ریستارت کردن بازی

¡ : اطلاعات

× : خروج از بازی

✓ : ثبت پاسخ

← : پاک کردن جواب

چگونه بازی کردن

هر بارابتدا نیاز دارید + را فشار دهید تا بازی آغاز شود. پس از فشردن + بازی به شما حروف یا تصاویری را با ترتیب اتفاقی به شما نشان خواهد داد که شما باید ترتیب دقیق نمایش آن ها را به خاطر بسپارید. سپس با استفاده از صفحه کلید مجازی موجود آن ها را با ترتیب قبلی وارد کنید. در پایان میبایست یا قشردن علامت تیک پاسخ خود را ثبت کنید. دقت داشته باشید که تنها سه بار کجاز هستیداشتباه کنید.
